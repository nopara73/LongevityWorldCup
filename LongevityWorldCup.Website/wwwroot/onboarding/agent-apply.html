Apply via AI Agent

Instead of filling out forms, you can apply to the Longevity World Cup through a conversational AI agent. The agent will guide you through entering your biomarkers, calculating your PhenoAge, and submitting your application.

 Option 1: Claude Code

If you have Claude Code installed, run the following command in the project directory:

/lwc-apply

Claude will walk you through the entire application process conversationally.

 Option 2: OpenClaw

Install the Longevity World Cup skill for OpenClaw:

openclaw skill install longevity-world-cup

Then start a conversation with your preferred messaging platform (WhatsApp, Telegram, Slack) and the agent will guide you through the process.

 Option 3: Manual Application

Prefer to do it yourself? No problem.
